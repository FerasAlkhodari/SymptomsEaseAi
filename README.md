# SymptomsEaseAi

مشروع لاستخدام الذكاء الاصطناعي لتشخيص الأمراض استناداً إلى الأعراض.

## نظرة عامة

هذا المشروع يستخدم تقنيات التعلم الآلي لتحليل الأعراض وتقديم تشخيص أولي للأمراض المحتملة. يتضمن المستودع:

- ملف Jupyter Notebook يشرح عملية تدريب وتطوير نموذج الذكاء الاصطناعي
- نموذج تم تدريبه
- أدوات التصور اللازمة لتحليل البيانات

## الملفات الرئيسية

- `Modified_EDA.ipynb`: دفتر Jupyter يوضح عملية استكشاف البيانات وإعداد النموذج
- `trained_model.h5`: النموذج المدرب
- `visualization_plots.py`: سكربت لإنشاء رسومات توضيحية للبيانات
